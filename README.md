# VRChat World Manager

VRChatで訪れたワールドや、行きたいワールドを管理するためのデスクトップアプリケーション。写真のメタデータから自動的にワールド情報を取得し、グループ分けや検索が可能です。VRChatアカウントを用いたログインが不要で使用できるのが特徴です。

## 特徴

### ワールド管理
- **自動インポート**: VRChatで撮影した写真をドラッグ&ドロップするだけで、メタデータからワールド情報を自動取得
- **手動追加**: ワールドのURLやIDから情報を取得して追加
- **グループ管理**: ワールドをカスタムグループに分類して整理
- **タグ管理**: ワールドに複数のタグを付けて分類（VRChat APIから自動取得も可能）
- **メモ機能**: 各ワールドに個人的なメモを追加
- **ログイン不要**: VRChatアカウント不要でワールド管理が可能

### 写真管理
- **PNG メタデータ解析**: VRChatで撮影したPNG画像からワールド情報を自動抽出
- **ワールドとの紐付け**: 写真を対応するワールドに自動的に関連付け
- **サムネイル表示**: ワールド詳細画面で関連する写真を表示

### 検索・表示機能
- **リアルタイム検索**: ワールド名、作者名、タグ、メモから高速検索
- **2つの表示モード**: グリッド表示とリスト表示を切り替え可能
- **詳細情報表示**: ワールドの説明、作者、タグ、写真などを一画面で確認

### ワールドオススメ機能
- **写真ディレクトリ監視**: VRChatの写真フォルダを設定すると、起動時に自動スキャン
- **新規ワールド提案**: まだ登録していないワールドを自動検出して提案
- **期間カスタマイズ**: 過去何日分の写真をスキャンするか設定可能（デフォルト14日）
- **無視機能**: 提案を無視したワールドは次回以降表示されない

## 必要な環境

- Node.js 18 以上
- npm または yarn

## インストール

```bash
# リポジトリをクローン
git clone [repository-url]
cd vrchat-world-manager

# 依存関係をインストール
npm install

# データベースをセットアップ
npx prisma generate
npx prisma db push
```

## 使い方

### 開発モードで起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

### 基本的な使い方

1. **写真を追加**: 「写真を追加」ボタンをクリックし、VRChatで撮影したPNG画像をドラッグ&ドロップ
2. **ワールドを手動追加**: 「ワールドを追加」ボタンから、ワールドのURLやIDを入力
3. **グループを作成**: サイドバーから新しいグループを作成し、ワールドを整理
4. **検索**: ヘッダーの検索アイコンから、ワールド名やタグで検索
5. **表示切り替え**: グリッド/リスト表示ボタンで見やすい形式に切り替え
6. **おすすめ機能**: ヘッダーの歯車アイコンから写真フォルダを設定すると、新規ワールドを自動提案

## 技術スタック

- **Electron**: デスクトップアプリケーションフレームワーク
- **React**: UIライブラリ
- **TypeScript**: 型安全な開発
- **Vite**: ビルドツール
- **Prisma**: データベースORM (SQLite)
- **Tailwind CSS**: スタイリング
- **VRChat API**: ワールド情報の取得
